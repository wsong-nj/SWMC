To use the pluginin "SWMC+.jar ", please follow the following steps:

1. Put the SWMC.jar File and the directory "extraLib" into the ProM project, then add all these jar file to the build path (see the picture g1.png).

2. Run the ProM project and you can search the "SWMC+" plugin-in (see the picture g2.png).

PS. You can use the log file in the "demo" as the input to test the SWMC+ plugin-in.
